Research Abstract

Background: Diagnostic delay reduces cancer survival rates by 10% each month. This project aims to predict cancer diagnosis through LLMs, NLP and Transfer Learning techniques to automate the extraction of necessary information from unstructured data: EHRs written in Spanish from the CANTEMIST dataset.

Methodology: The data was labeled with LLM models - GPT-4 was chosen over Google Gemini, then subsequently used to train the SpaCy NER model in Python, to apply Machine Learning to the entities extracted to classify the diagnosis, comparing to the ground truth with an F1 of 0.97.

Key Findings

  • High Performance: Achieved 0.97 F1-score with 97% accuracy on cancer classification
  • LLM Effectiveness: GPT-4 outperformed Google Gemini for medical text annotation
  • Cost-Effective Approach: Eliminated need for expensive medical expert annotations
  • Spanish Language NLP: First generalised system for Spanish clinical text analysis
  • Transfer Learning Success: Demonstrated effective knowledge transfer from LLMs to traditional ML

Model Optimization Techniques

  • TF-IDF Vectorization: Converted clinical text to numerical features weighted by term importance
  • Feature Selection: Used Chi-squared test to select most informative features for cancer prediction
  • Multiple Algorithms: Grid search across Logistic Regression, Random Forest, and Gradient Boosting
  • Comprehensive Evaluation: Cross-validation, confusion matrices, feature importance analysis

Future Work & Limitations

  • Validation with medical expert annotations for NER evaluation
  • Extension to other languages and medical domains
  • There is a lack of publicly available data for clinical texts, so the results may not generalise to other datasets
  • Exploration of more advanced LLMs and hybrid models
  • Larger-scale evaluation on diverse patient populations
