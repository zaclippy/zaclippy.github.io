About the Project

This project showcases the integration of Natural Language Processing (NLP) techniques for cancer diagnosis support. It utilizes a custom-trained Named Entity Recognition (NER) model to analyze clinical texts and identify relevant medical entities.

The application leverages advanced machine learning algorithms to classify clinical notes, providing healthcare professionals with valuable insights to assist in diagnosis.

Technologies Used

  • LLM: GPT, Gemini, Claude, LLAMA
  • HTML & JS
  • Spacy for NLP
  • Python-Jupyter
  • Machine Learning Models
  • RESTful API

Future Work

Research results will be uploaded to this website
